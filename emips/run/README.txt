Data download
----------------

MEIC:
http://meicmodel.org/
CAMS:
https://ads.atmosphere.copernicus.eu/cdsapp#!/dataset/cams-global-emission-inventories?tab=overview
HTAP:
https://eccad3.sedoo.fr/
NMVOC speciation data:
http://iek8wikis.iek.fz-juelich.de/HTAPWiki/WP1.1

Read before running
----------------------

1, The scripts in "run_meic_cams_htap" directory is used for regular processing, and the scripts in "for_xxx" directory is used for specific processing to meet the requirements of each model. 

2, You need to place the files in the "Shapefile" directory into the "map" directory of MeteoInfo before running.
