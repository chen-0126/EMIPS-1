Data download
----------------

MEIC:
http://meicmodel.org/
CAMS:
https://ads.atmosphere.copernicus.eu/cdsapp#!/dataset/cams-global-emission-inventories?tab=overview
HTAP:
https://eccad3.sedoo.fr/
NMVOC speciation data:
http://iek8wikis.iek.fz-juelich.de/HTAPWiki/WP1.1

Read before running
----------------------

1, Modify the data path in the "emission_xxx.py", "run_voc.py", "total_run_xxx.py", "merge_meic_cams_htap_tw.py" 
    and "run_all.py".

2, Set parameters(chemical mechanism, grid projection) in "run_all.py".
